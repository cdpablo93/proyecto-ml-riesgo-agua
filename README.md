# proyecto-ml-riesgo-agua
Proyecto elaborado para el diplomado en ciencia de datos de la ENES unidad Leon. EL proyecto consta de un modelo de machine learning supervisado entrenado con un dataset de 6mil muestreos de calidad de agua en la ciudad de méxico y que es capaz de estimar si una ubicacion geográfica podria representar riesgo microbiológico en la calidad de agua.
